# 如何更新網站並自動部署

每次您修改完程式碼 (例如 `index.html`) 並存檔後，請依照以下步驟將修改上傳，Vercel 就會自動更新網站。

---

## 方法一：使用 VS Code 介面 (最簡單，推薦！)

1. **修改並存檔**：確認您的檔案已經儲存 (`Ctrl + S`)。
2. **打開原始碼控制**：點擊左側邊欄的 **Source Control** 圖示 (通常是第三個，看起來像一個分岔的路線圖)。
3. **輸入備註**：在輸入框中輸入這一次修改了什麼 (例如：「修改標題文字」)。
4. **提交 (Commit)**：點擊輸入框上方的 **Commit** 按鈕 (或是打勾圖示)。
   - *如果系統問您是否要暫存所有變更 (Stage all changes)，請選「是 (Yes)」。*
5. **同步 (Push)**：點擊藍色的 **Sync Changes** (同步變更) 按鈕。
   - *這會將您的修改上傳到 GitHub。*

---

## 方法二：使用終端機 (Terminal) 指令

如果您喜歡打字，也可以在下方的 Terminal 輸入以下三行指令：

1. **加入檔案**：
   ```bash
   git add .
   ```

2. **提交並加入備註**：
   ```bash
   git commit -m "這裡寫下您的修改備註"
   ```

3. **推送到 GitHub**：
   ```bash
   git push
   ```

---

完成以上任一種方法後，稍等約 1~2 分鐘，您的網站就會自動更新內容囉！
